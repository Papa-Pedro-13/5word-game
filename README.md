# Игра "5 букв"

Учебный проект игры на React, также была реализована версия мобильного приложения на react Native.

<h2><b>Стек frontend технологий:</b> React, TypeScript, TailwindCSS, ReactNative.</h2>

<h3>Инструкция по запуску:</h3>

```
npm install
npm run dev
```
